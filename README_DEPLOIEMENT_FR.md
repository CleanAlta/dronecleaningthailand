# CleanAlta Thailand – Pack prêt à publier

## Ce pack contient
- le site trilingue dans l’ordre de priorité **EN / TH / FR**
- les pages services
- les pages villes prioritaires : **Bangkok, Koh Samui, Krabi, Chiang Mai**
- les pages secteurs
- les pages références projets
- le formulaire de contact / devis
- les balises SEO, hreflang, sitemap, robots et données structurées

## Domaine retenu
**dronecleaningthailand.com**

## Positionnement public
Le site est publié sous la signature **CleanAlta Thailand**.
Le domaine reste orienté acquisition locale et compréhension immédiate de l’offre.

## Mise en ligne
1. Dézipper le pack
2. Déployer sur Netlify, Vercel ou Cloudflare Pages
3. Pointer le domaine `dronecleaningthailand.com`
4. Vérifier les formulaires, Search Console et l’indexation du sitemap

## Vérifications avant publication
- confirmer les numéros France / Suisse
- valider l’adresse email de contact
- remplacer ou compléter les visuels si nécessaire
- connecter Google Analytics 4 et Google Search Console
- créer et optimiser la fiche Google Business Profile si une adresse locale réelle est disponible

## Point important
Le site mentionne une **recherche de partenaires pour l’ouverture du bureau en Thaïlande**.
Il ne prétend pas disposer d’un bureau local déjà ouvert.
