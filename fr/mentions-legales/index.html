Éditeur du site

CleanAlta Thailand est présenté en ligne comme le site commercial thaïlandais des services CleanAlta. La référence groupe utilisée dans le package est TELEPILOTE SAS, avec les actifs de marque CleanAlta et le contact eco@cleanalta.net.

Avant mise en ligne définitive, remplace les éléments juridiques provisoires par l’entité d’exploitation finale en Thaïlande, l’adresse locale, les références fiscales et les données d’hébergement.

Contact et responsabilité éditoriale

La responsabilité éditoriale doit être attribuée à l’opérateur de production final avant lancement.

  • Contact principal : eco@cleanalta.net
  • WhatsApp : +33 6 86 50 55 71

Propriété intellectuelle

La structure du site, les textes, la marque, les photos internes et le code sont fournis pour le déploiement CleanAlta.

Le package final n’utilise aucun visuel hotlinké. Toutes les images référencées sont stockées localement dans le dépôt.

Hébergement et déploiement

Ce package est statique et se déploie sur GitHub Pages, Netlify, Cloudflare Pages ou tout autre hébergeur statique.

L’hébergeur, le DNS, le nom de l’éditeur légal et les politiques doivent correspondre à l’opérateur commercial final au lancement.
